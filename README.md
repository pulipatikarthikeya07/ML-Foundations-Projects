# Machine Learning Foundations Projects

This repository contains practical implementations of foundational machine learning algorithms using Python and Scikit-Learn.

## 1. Predictive Real Estate Pricing Engine
* **Type:** Supervised Learning (Regression)
* **Objective:** Predict residential property prices based on features like square footage, bedrooms, and property age.
* **Key Frameworks:** Pandas, Scikit-Learn (Linear Regression, StandardScaler)

## 2. Customer Churn Analytics System
* **Type:** Supervised Learning (Classification)
* **Objective:** Analyze user behavior patterns to predict consumer churn risk.
* **Key Frameworks:** Scikit-Learn (Decision Tree Classifier)
